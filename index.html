My Projects and Designs

UX-Design Group Project

This project was created as a group, focused on empowering Indigenous youth. We applied the full UX design process, from research, to wireframe development, to coming up with an end result. The goal was to identify user needs, analyze pain points, and design a solution that could help to empower Indigenous youth through online leadership opportunities.

Our group project began with user research, including conducting a survey and creating personas to better understand participant needs. Insights gathered during the research phase formed the creation of low and high-fidelity wireframes.

The end result was a tailored virtual conference platform. The platform allows the user to register for a LIVE workshop, download pre-recorded material, and connect with a mentor.

UX-Design The Digital Commute Project

This project was developed as a solo project. I utilized the full UX design process from research analysis to persona development. Based on the research and insights gatheres I created low-fidelity wireframes to map out my ideas and then turned those into high-fidelity wireframes to better showcase the designs.

The end result was an app called "The Digital commute." The Digital Commute is an ap that helps the user disconnect at the end of the day. It allows them to catalogue their achievements for the day and set goals for the next day. It provides an opportunity for a transition from "work time" to "personal time"

Web-Development - Capstone Project

This project involved the design and development of a responsive multi-page website using a clean, single column layout to provide users with a clear and focused browsing experience. The objective was to create an intuitive website structure that prioritized readability, accessibility, and seamless navigation across all devices. The website was developed using VS Code and organized into multiple pages, each dedicated to specific content and functionality.

The development process included implementing responsive layouts, accessible navigation menus, interactive elements, and color theory composition throughout the site. Careful attention was given to usability, to ensure a reliable and engaging experience.

The completed multi-page website consists of a single column layout, responsive design, and consistent navigation structure. The project demonstrates my ability to develop organized, user-focused websites that balance functionality, performance, and visual appeal.
